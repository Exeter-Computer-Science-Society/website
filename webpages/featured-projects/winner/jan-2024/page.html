Spotify Album Art Display - Thomas Newbold

A small flask web-app which displays album art - or playlist art - for the currently playing track on the linked Spotify account

Uses the Spotify Web API (https://developer.spotify.com/documentation/web-api).

It can be run as localhost in the browser, but I wanted something more physical and interactive, so I got the server and webpage running standalone on Raspberry Pi Zero with a 3.5" touchscreen interfaced with the GPIO pins, which now sits on my desk.
