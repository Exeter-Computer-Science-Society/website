title: Apply =====================

Feature your project!


To have your project featured please first read the following!

What projects qualify?

Any computer science related project that you you have been working on are working on (It does not need to be completed!) and want to be recognised for and let others know about.

Bonus points if your project is Open Source and is useful for others!


How can I apply with my project?

Email excs@groups.exeterguild.com with a message using the following format:


To: excs@groups.exeterguild.com
Subject: Featured Project Submission
{Project Name}

- {A description about your project! This will be used on the website if your project wins}

- Optional: {An image/images of your project that help to tell people about it!}

- Optional: If the project is open source, link to the github repo!

- Other Optional Info:

  • Your Name
  • Links to any socials you wouldn't mind shared
  • An image of yourself

*If your project is selected, you consent to all the information you have provided in this email to be published on our website and our socials.
